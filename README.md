# wheava

## Run locally
Put the following in a new file `src/strava/strava-config.ts`
```javascript
export const STRAVA_CONFIG = {
    clientId: '<YOUR_STRAVA_CLIENT_ID>',
    clientSecret: '<YOUR_STRAVA_CLIENT_SECRET>',
}
```

Then run `npm install`
